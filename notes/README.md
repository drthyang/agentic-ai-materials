The thinking layer of this project — strategy, positioning, and study
materials. Code answers "how"; this folder answers "why" and "toward what".

| File | What it holds |
|---|---|
| [impact-strategy.md](impact-strategy.md) | Field map, where individual leverage is, ranked impact paths, pitfalls, 90-day arc |
| [positioning.md](positioning.md) | How this project presents the full skill stack; visibility plan and artifact checklist |
| [reading-list.md](reading-list.md) | Curated papers/resources, themed, with why-read and when-to-read |

Convention: these are living documents — date significant edits inline, keep
superseded thinking as strikethrough or a "revisions" note rather than
deleting it (the evolution of the strategy is itself part of the record).
